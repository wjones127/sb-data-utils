Creates a new variable in dataset that gives the cumulative value based on another variable giving growth counts.
Parameters:

output string the name of the property to save the cumulative counts in.
baseline numeric (optional) the intial value of the cumulative value before any growth.
Source:

Type

         {'count': 3}];

(static) fill(data, input, defaults, start, end) → {array}

Fills in missing values in a data set, based on a variable with integer values. defaults gives the values to fill in missing values for other variables.
Parameters:
Name Type Description
data array the data set to fill in.
input string the name of the property to be filled in. Needs to be integer values.
defaults object an object with the default values for the other variables.
start number (optional) a number giving where the range should start. If this number is larger than the max value of existing values, it will be used instead as the upper bound.
end number (optional) a number giving where the range should end.
Source:
  • data-utils.js, line 112
Returns:
Returns the new data set with filled in observations.
Type
array
Example
var data = [{'year': 2010, 'likes': 12},
            {'year': 2011, 'likes': 26},
            {'year': 2014, 'likes': 8}];
var defaults = ['likes': 0];
require('data-utils').fill(data, 'year', defaults, 2015);
// returns [{'year': 2010, 'likes': 12},
//          {'year': 2011, 'likes': 26},
//          {'year': 2012, 'likes': 0},
//          {'year': 2013, 'likes': 0},
//          {'year': 2014, 'likes': 8},
//          {'year': 2015, 'likes': 0}];

(static) percentGrowth(data, input, output, trim)

Saves the percent growth in a new variable in a dataset, calcualted based on a cumulative count variable.
Parameters:
Name Type Description
data array an array of objects to be used as the data set.
input string the name of the property giving the changing count.
output string the name of the property to save the percentages in.
trim boolean where to trim off the first observation (otherwise it will just be saved as zero.)
Source:
  • data-utils.js, line 192
Returns:
Returns a new data set with the new percent growth variable added.
Example
var x = [{'count': 10},
         {'count': 15},
         {'count': 30}];
require('data-utils').percentageGrowth(x, 'count', 'growth');
// returns [{'count': 15, 'growth': 50},
//         {'count': 30, 'growth': 100}];
